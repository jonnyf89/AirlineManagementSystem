# AirlineManagementSystem
A journey and fuel management system using Python.

Inputs: 
1. Aircraft code
2. Starting airport code
3. 2-4 additional airport codes

Outputs:
1. Range of selected aircraft
2. Suggested, most economical route between selected airports
3. Total journey distance
4. Number of refuels required
5. Airport(s) with the cheapest fuel
6. Price of fuel at each selected airport
7. If inputted journey is impossible in selected aircraft

Airport and aircraft data comes from csv files which are included in the repositry.
Distance between airports is calculated by the programme in the AirportAtlas class.
Extensive error handling and unit testing.

There is a supporting document included in the repositry which made up part of the submission for this project.
